# cygnus-vn-erp
Um centro operacional centralizado que unifica a gestão financeira, o estoque, as vendas e a contabilidade em um único fluxo de informações de negócios de alta densidade.

Sistema ERP modular para gestão de:

- Vendas
- Estoque
- Compras
- Financeiro
- Relatórios
- Contabilidade

Sistema em construção.
